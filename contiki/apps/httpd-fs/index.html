Welcome to the Contiki web server!

The web pages you are watching are served by a web server running under the Contiki operating system on an unexpanded Commodore 64 equipped with a custom built Ethernet card.

The screenshot above shows what the desktop looks like on the C64 on which the web server is running. Please note that this web page, the screenshot, the graphical border around the menu to the left, and the scripts that generate the dynamic pages for file statistics, the currently open connections and the system processes all are served by this Contiki web server.

For more information about the Contiki operating system, see the Contiki web site:

http://dunkels.com/adam/contiki/

For pictures of the Commodore 64 on which this server is running, see the "Pictures" section of the TFE (The Final Ethernet cartridge) homepage:
