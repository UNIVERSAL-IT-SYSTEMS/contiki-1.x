Welcome to the Contiki demo server!

The web pages you are watching are served by a web server running under the Contiki operating system on an an Ethernut embedded Ethernet board, which consists of a 14 MHz AVR Atmega128 microcontroller with 32 kilobytes of RAM and 128 kilobytes of flash ROM, and a RealTek RTL8019AS Ethernet chip.

http://contiki-demo.sics.se/

Remote display

It it possible to remotely access the Contiki desktop of this server directly from your web browser! See the remote display page.

Memory usage

The entire Contiki AVR system uses slightly more than 20 k RAM, of which 6 k is used for the virtual display 8 k for the web browser. All code and these web pages are stored in the microcontroller's on-chip flash ROM.
